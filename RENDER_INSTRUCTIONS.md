# Render'ga Joylash Yuborish Qo'llanmasi

Siz Render.com saytida "New Web Service" ni qo'lda ochayotgan bo'lsangiz, sahifadagi maydonlarni quyidagicha to'ldiring:

1. **Name**: `Shifokor`
2. **Language**: `Node`
3. **Branch**: `main`
4. **Root Directory**: *(bo'sh qoldiring!)*
5. **Build Command**: `npm run build`
6. **Start Command**: `npm start`
7. **Instance Type**: `Free` (bepul)

### Environment Variables (Atrof-muhit o'zgaruvchilari)
Pastki qismga tushib **"Add Environment Variable"** tugmasini bosing va quyidagilarni kiriting:

*   **NAME**: `NODE_VERSION`
    **VALUE**: `20`

*   **NAME**: `BOT_TOKEN`
    **VALUE**: *(o'zingizning bot tokeningizni kiriting)*

*   **NAME**: `GROQ_API_KEY`
    **VALUE**: *(o'zingizning Groq API kalitingizni kiriting)*

Barchasini kiritib bo'lgach **"Create Web Service"** tugmasini bosing! 🚀
Loyihangiz serverda to'liq ishga tushadi va aslo uxlab qolmaydi.
